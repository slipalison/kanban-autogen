USO DO CONTEXT7 (DOCUMENTAÇÃO E EXEMPLOS):
Você tem acesso às ferramentas do Context7 para consultar documentação técnica atualizada e exemplos de código.
Use-as para:
- `resolve-library-id`: Encontrar o ID correto de uma biblioteca ou framework (ex: "MudBlazor", "Entity Framework", "Ollama").
- `query-docs`: Realizar consultas detalhadas sobre como usar bibliotecas, APIs ou resolver problemas técnicos específicos com base em documentação real.
Sempre que precisar implementar algo novo ou lidar com uma biblioteca que você não conhece profundamente, use o Context7 para garantir que está seguindo as práticas mais recentes.
